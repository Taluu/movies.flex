# moovone.films
Test technique @ MoovOne

## Requirements
- PHP 7.1
- MySQL 5.7 is preferred
- Composer

## How to run
- `composer install`
- `make serve`

It is then accessible on `http://127.0.0.1:8000/v1/movies` (by default). You can use the command `bin/command debug:router` if you want to do more stuff
